🧙‍♂️ Quiz do Chapéu Seletor (The Sorting Hat Quiz)

Este é um projeto simples e divertido que eu pensei em fazer por ser fã da franquia de livros do Harry Potter que simula a icônica cerimônia do Chapéu Seletor. Responda as três perguntas e descubra a qual casa de Hogwarts você pertence! 🏰✨

📋 Sobre o Projeto

O programa interage com o usuário através do terminal, fazendo três perguntas de múltipla escolha. Dependendo da resposta escolhida (utilizando entradas numéricas), pontos específicos são distribuídos entre as quatro casas de Hogwarts:

  🦁 Gryffindor (Grifinória)
  🐍 Slytherin (Sonserina)
  🦅 Ravenclaw (Corvinal)
  🦡 Hufflepuff (Lufa-Lufa)

No final, o programa calcula qual casa obteve a maior pontuação e revela o resultado ao usuário.
